# 
Instalación y ejecución del proyecto
Paso 1: Clonar el repositorio
git clone <URL-del-repo>
Paso 2: Entrar a la carpeta del proyecto <lista-compras>
Paso 3: iniciar el proyecto
bun start
Paso 4: escanear el qr y entrar con expo go 
Paso 5: como utilizarlo
- Escribí un producto y tocá **Agregar** o **Enter**.
- Tocá un ítem para **marcar/desmarcar** (hecho).
- **Dejá apretado** un ítem para **eliminarlo**.
